Установка Fallout Extended Edition

Спасибо за проявленный интерес! Данная страница поможет Вам с установкой.

Перед началом

Требования
  • Запакованная 15 GB, распакованная 32 GB
  • Windows 7 64-bit и выше
  • Обновленные драйвера видеокарты
  • Visual C++ пакеты
  • DirectX Runtime
  • .NET 6.0
Перед установкой
  • DLC присутствуют
  • Достижения работают
  • После установки обязательна новая игра
  • Не ставить в системные папки
  • Не накатывать поверх оригинальной игры
  • В пути не должно быть кириллицы

Запуск и настройка установщика

  • Качаем торрент, запускаем программу установки.
  • Выбираем путь куда будет установлена сборка, жмем далее.
    1. В нем не должно быть русских символов
    2. Нельзя ставить в Program Files и Program Files (x86)
    3. Нельзя ставить поверх оригинальной игры
  • На следующей странице отмечаем желаемые параметры.
    • Дополнительные библиотеки
      Нужны для корректного запуска и установки сборки.
      Обязательно устанавливаем.
    • DXVK
      Перевод игры с DirectX 9 на Vulkan.
      Сильно улучшает производительность, но может вызвать баги и вылеты.
      Если решите поставить, то убедитесь, что видеокарта поддерживает его и драйвера обновлены, иначе игра перестанет запускаться.
      Так же вместе с ним советую включить полноэкранный режим.
      DXVK 2.0 при запуске часто вылетает.
      Для DXVK 2.0 нужна поддержка Vulkan 1.3
      Для DXVK 1.10.3 нужна поддержка Vulkan 1.1
      Отключается удалением d3d9.dll из папки сборки.
    • ReShade
      Делает картинку игры более сочной. Скриншоты для сравнения.
      • Указываем путь к "FalloutNV.exe" сборки через "Browse"
      • Выбираем DirectX 9, если ставили DXVK выше, то Vulkan.
      • Указываем путь к "Vibrance.ini" в папке сборки через "Browse"
      • На странице с эффектами жмем "Next"
      • Ждем окончания загрузки, жмем "Finish"
  • Жмем установить и ждем окончания.
    • Удаление папок New Vegas'a связанных с MO
      Во время установки может вылезти подобное окно, говорящее что нужно удалить все профили MO связанные с New Vegas'ом.
      Удаляются они из Global Instance для корректного определения пути к сборке для МО.
      Вводим номер папки связанной с Vegas'ом и жмем Enter. Продолжаем так делать до той поры пока не останется папок связанных с Vegas'ом, после этого вводим "0", жмем Enter.
      Если папок вообще не останется, то установщик автоматически перейдет к следующим шагам.

Выбор пресета

  • Запускаем "New Vegas EE Launcher" с рабочего стола.
  • Выбираем желаемый профиль в верхней части МО.

  • Модули

      • БАЗА - дополнительный и восстановленный контент, звуки, погода, интерфейс, анимации, багфиксы, оптимизация.
      • ГРАФИКА - текстуры и модели.
      • ГЕЙМПЛЕЙ - геймплейные возможности, корректировка баланса, усложнение игры.

    Пример смены профиля

Восстановление порядка загрузки

  • Запускаем через кнопку "Запустить".
  • Настраиваем, обязательно ставим качество текстур на высокое, закрываем.
      • Советую поставить высокий пресет графики и отключить тени.
      Они все равно есть только у NPC.
  • Восстанавливаем порядок загрузки.
  • Включаем по желанию опциональные моды.
    • Так же имеется возможность выбрать пип-бой и настроить твики.
  • Закрываем МО, запускаем "New Vegas EE" с рабочего стола, наслаждаемся.
